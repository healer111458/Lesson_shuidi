- 页面先写结构
  类名 .key>.sound 一个盒子
  从外到内。
  良好的结构是页面的基础
- 写结构时不用考虑样式
- 样式
 - 块级元素 block
   行类元素
 - 弹性布局
   display:flex;
   所有子元素的块级能被干掉。
   min-height：100vh;相对高度
   justify-content:center;
   align-items:center;
 - html 默认的字体是16px；
   rem 跟vh 一样都是相对单位。
   rem相对于html字体大小
   0.4rem = 16px*0.4=
   6.389px
   在手机端px不准确
   作用：适配所有手机
- 盒子模型
  margin+border+padding+content
